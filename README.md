# Мониторинг экосистемы: классификация звуков птиц

Проект для автоматического мониторинга экосистемы через анализ звуков птиц с использованием машинного обучения.

## Описание

Система собирает аудиозаписи птиц, классифицирует их с помощью предобученной нейросети и генерирует оповещения о важных событиях в экосистеме.

### Возможности

- Сбор аудиозаписей птиц с открытого API Xeno-Canto
- Классификация звуков с помощью предобученной модели
- Оценка качества распознавания (accuracy, precision, recall, f1-score)
- Визуализация результатов (графики, confusion matrix)
- Система оповещений об аномалиях и редких видах

### Типы оповещений

| Тип | Описание |
|-----|----------|
| RARE_SPECIES | Обнаружен редкий или охраняемый вид |
| LOW_CONFIDENCE | Модель не уверена в предсказании (< 50%) |
| UNKNOWN_SOUND | Возможно неизвестный вид или шум (< 30%) |
| HIGH_FREQUENCY | Аномально высокая активность вида |
| LOW_FREQUENCY | Подозрительно низкая активность вида |

## Использование

1. Открыть `practice_project.ipynb` в Google Colab или Jupyter Notebook
2. Запустить все ячейки по порядку
3. Результаты классификации сохранятся в `bird_classification_results.csv`
4. Оповещения сохранятся в папку `alerts/`

## Технологии

- **Python 3**
- **Transformers** — загрузка предобученной модели с Hugging Face
- **librosa** — работа с аудио
- **pandas** — обработка данных
- **scikit-learn** — метрики качества
- **matplotlib / seaborn** — визуализация

## Модель

Используется предобученная модель [dima806/bird_sounds_classification](https://huggingface.co/dima806/bird_sounds_classification) из Hugging Face Hub.

## Источник данных

Аудиозаписи загружаются с [Xeno-Canto](https://xeno-canto.org/) — открытой базы звуков птиц со всего мира.
